## 一、产物总览

本项目共包含 **27 个文件**，分为四大类：

### 1. 官方电商网站（中英双语）
| 文件 | 用途 |
|------|------|
| `index.html` | 首页：Hero、三档产品、配件商城、特性、规格、对比表、场景 |
| `cart.html` | 购物车页 |
| `checkout.html` | 结算页 |
| `css/style.css` | 全站样式 |
| `js/i18n.js` | 中英文双语文案库 |
| `js/cart.js` | 购物车逻辑（读取 catalog） |
| `js/store.js` | **商品目录中枢**：从 API 拉取商品，动态渲染产品/配件，应用促销价、上下架 |
| `server.js` | **电商后端（Node.js 零依赖）**：REST API + 静态托管 + 管理后台支持 |
| `data/products.json` | **商品数据源**（单一事实源）：含价格、上架状态、促销政策 |
| `admin.html` | **商家管理后台**：产品上下架、价格管理、促销政策、增删商品 |

### 1.1 完整电商能力
- 产品上下架（`active` 开关，下架即不在官网展示）
- 价格管理（随时改价，官网实时生效）
- 促销政策（折扣 % / 立减 $，带标签，前端显示划线价 + 促销徽章）
- 商品增删（admin 后台）
- 16 个商品（3 整机 + 1 升级套装 + 12 配件），均可独立下单

## 二、本地预览

```bash
# 在项目根目录启动本地服务器
cd /Users/weigan/WorkBuddy/2026-07-17-15-42-02
/Users/weigan/.workbuddy/binaries/node/versions/22.22.2/bin/node -e "
const http=require('http'),fs=require('fs'),path=require('path');
const root=process.cwd(),mime={'.html':'text/html;charset=utf-8','.css':'text/css','.js':'application/javascript','.png':'image/png'};
http.createServer((q,s)=>{let u=q.url==='/'?'/index.html':q.url.split('?')[0];
fs.readFile(path.join(root,u),(e,d)=>{if(e){s.writeHead(404);s.end('Not found')}else{s.writeHead(200,{'Content-Type':mime[path.extname(u)]||'application/octet-stream'});s.end(d)}})}).listen(8090,()=>console.log('http://localhost:8090'))"
```
