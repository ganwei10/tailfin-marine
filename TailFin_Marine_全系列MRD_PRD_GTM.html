Part 1 · MRD 市场需求文档 Market Requirements

1.1 市场背景与趋势

全球休闲垂钓持续结构性增长：装备向轻量化、便携化、模块化演进；智能与电动化渗透率提升；环保可持续（可降解、防穿刺、锁鲜）成为购买要素；「钓鱼 + 露营 + 文旅」融合催生「钓后处理（Clean & Cook）」新消费场景。现有市场以被动式折叠去鳞台（HDPE 台面）为主，动力去鳞、真空锁鲜与离网供电长期割裂——TailFin 首创「动力模块化钓后处理系统」占据品类空白。

1.2 三场景需求定义

皮卡 / 陆地（基准）
Job：停好车在尾门把鱼处理完。供电：12V 点烟器 / 内置电池。审美：粗犷实用、扛造。约束：车载收纳、防脏车内。
钓鱼船 / Vessel
Job：在船甲板把鱼处理完、上岸即冷冻。供电：12/24V 船电 DC。审美：同粗犷实用。约束：盐雾、摇摆、舷外排水合规、电气安全。
游艇 / Yacht
Job：自己甲板上的 Catch-to-Table。供电：110/220V AC + 12/24V DC 多电压。审美：颜值、静音、隐藏收纳。约束：奢华外观、低噪、与内饰协调。

1.5 市场机会（规模支撑）

  • 全球休闲垂钓装备 2025 约 $14.8B → 2034 $24.4B（CAGR 5.7%，Dimension）；广义渔具 2025 $16.5B → 2032 $22.08B（CAGR 4.25%）。
  • 美国 2024 年 5,790 万钓鱼人口（创纪录，RBFF）；欧洲 870 万海钓者、年支出 €59 亿；西澳 75 万钓客、$24 亿澳元经济；中国 1.4 亿钓友、轻量化装备 CAGR 17.3%。
  • 三场景 TAM/SAM/SOM（估算）：TAM=全球钓具 $24.4B；SAM=便携钓后处理设备（陆地+船+游艇）≈ $3–4B；SOM（3 年目标）≈ $80–120M（以分级渠道渗透计）。

2.2 平台化架构（跨场景复用内核）

统一基础平台 = 中央仓双槽叠层（Pogo Pin 盲插） + 五大即插即用模块：动力去鳞机（12V 有刷 / 24V 无刷，316 不锈钢刀头）、电动鱼刀（12V 往复，防水航空插口）、真空密封 cartridge（干湿两用 ≥-75kPa）、24V 电池包（离网 ≥2h，BMS 双充）、HDPE 食品级砧板。三场景共用此内核，差异仅在「环境适配层」。

2.3 三场景 PRD 与 BOM 改动

2.3.1 皮卡 / 陆地版（已交付，基准 BOM）

  • 箱体 ABS+PC 80×44×26cm；12V 100W 有刷去鳞机（Standard）/ 24V 200W 无刷（Pro）；循环给水 5L/min；接点烟器 12V 或 24V 电池；316 刀头、HDPE 砧板。
  • 认证：消费电子级（FCC/CE 适用项），无船规要求。

2.3.2 钓鱼船 / Vessel 版（下一刀，P0 优先）

P0 上船门票：电气船规化（直连 12/24V 船电、防反接/低压保护，ABYC E-11）；IP67 接线盒与密封（3m×30min 不进水）；集水槽 + 舷外排水合规（MARPOL 惯例）；抗摇摆快装导轨夹具（适配 Ø22–32mm 栏杆）。
P1 差异化：壳体 ASA+316 骨架（盐雾 96h）、ABYC/CE 认证、抗摇摆锁扣、接地绝缘。
P2 选配：舷边排水泵适配器、316 砧板、夜航 LED、船电智能监测。

2.3.3 游艇 / Yacht 版（高端衍生，建在 Vessel 之上）

奢华外观（新开模）：柚木纹 / 拉丝不锈钢面板，与游艇内饰协调；隐藏走线结构。
多电压自适应：新增 110/220V AC 转换电源 + 12/24V DC，全球码头/船电通吃（这是相对陆地版最大的新增件）。
静音与减震：静音电机版本 + 橡胶减震脚，避免打扰舱内客。
折叠收纳：可收入游艇储物柜；氛围灯等精致配件。

Part 3 · GTM 上市策略 Go-To-Market

3.1 分阶段路线图（纵向铺场景，横向铺渠道）

阶段 0 · 陆地已交付：Mini/Standard/Pro/Kit 在售，Amazon + 自营站 + 露营/钓具渠道跑通。
阶段 1 · 钓鱼船优先（W1–20）：EVT→船长种子 Beta→认证→量产；先打 Charter 船长滩头。
阶段 2 · 渠道与 OEM：海事分销商（West Marine/Fisheries Supply）、国内海钓店/游艇会、船厂 OEM 前装试点。
阶段 3 · 游艇衍生：船版量产后启动 Yacht（奢华层 + AC），走游艇会/高端海事渠道。
阶段 4 · 品牌出海：展会（METSTRADE/IBEX/中国渔博会/悉尼船展）+ 内容 + 三级分销全球化。

3.2 渠道策略（分场景侧重）

场景主渠道打法
皮卡露营/钓具连锁、Amazon、户外电商大众内容 + 性价比走量
钓鱼船海事分销商、船具店、船厂 OEM、船长 KOL滩头船长种子包换内容与口碑
游艇游艇会、高端海事零售、定制船厂奢华体验 + 私域/会员

3.3 定价策略（三场景 × 三级渠道）

沿用三级分销：授权零售 70% / 区域经销商 60% / 国家分销 50% MSRP；含 MAP 价格保护、区域独家、阶梯返点 3–8%、认证售后支持。三场景 MSRP 见 §2.1 矩阵；船版/游艇版溢价由「船规电气 + 合规认证 + 奢华外观」支撑，毛利结构见 Part 4。

3.4 内容与品牌

  • Charter 船长种子包：寄 Vessel Pro 样机 + Kit + 备用刀头 + 真空袋；30 天实海试换 3 段视频 + 反馈表，达标样机归船长并享首批量分销权。
  • 话术分众：Charter「几趟回本」/ 休闲船东「甲板上的 Catch-to-Table」/ 游艇「你游艇上的米其林体验」/ 合规「船规级、ABYC/CE，敢上海」。
  • 展会：METSTRADE（阿姆斯特丹）、IBEX（美）、中国渔业博览会、悉尼船展实机演示。
