结论先行：钓鱼船版是现有皮卡版（陆地版）最自然的海洋延伸——核心电子、电机、电池、真空、刀头、中央仓全部 100% 沿用（本就带 316 不锈钢 + 防水航空插口 + 24V 电池基因），仅需「材料加固 + 上船附件 + 电气合规」三类改动，BOM 复用率约 80%。建议优先于游艇版推进，并同步沉淀为「Vessel 平台」，未来游艇版仅是在此之上加 AC 适配器 + 奢华外观的下游变体。

一、产品定义与 SKU 规划 Product Definition & SKUs

钓鱼船场景的真实 Job-to-be-done 与皮卡一致：「在船上把鱼处理完、上岸即冷冻」。差异只在供电（船电 12/24V DC 而非点烟器/电池）与受力环境（盐雾、摇摆、舷外排水合规）。据此规划三个 SKU：

Vessel Core · 船规基础版

建议 MSRP $599

基于 Standard（$499）改。面向休闲渔船东、小型 charter、海钓俱乐部。保留 12V 100W 去鳞机 + 循环给水 + HDPE 砧板，新增导轨夹具、集水槽排水、防反接模块、IP67 接线盒。

Vessel Pro · 船规旗舰版

建议 MSRP $1,149

基于 Pro（$999）改。面向专业 charter、商业渔船、游艇钓台。24V 无刷去鳞 + 电动鱼刀 + 中央仓真空 + 24V 电池离网，全船规化（316 框架、盐雾认证、双充船电）。

Vessel Kit · 船用升级套件

给已购 Standard / Pro 的存量用户「上船」。含导轨/夹具支架 + DC 直连线束 + 防反接/低压保护模块 + 集水槽舷外排水套件 + 316 紧固件包。建议 MSRP $149。

复用定位

Vessel 不是新品类，是平台的一个「环境适配层」。陆地版 / 船版共享同一中央仓、同一套模块（去鳞/刀/真空/电池/砧板）。未来此层再叠加「游艇奢华皮肤 + AC 适配器」即成本最低的高端变体。

五、GTM 上市方案 Go-To-Market

5.1 滩头：Charter 船长种子包（最低成本拿内容+口碑）

寄样清单（每套）：Vessel Pro 样机 1 台 + Vessel Kit 1 套 + 备用 316 刀头 3 片 + 防穿刺真空袋 20 片 + 品牌贴纸/旗 + 使用卡。

合作话术要点：「免费上船试用 30 天；您只需在出海带客时自然使用、拍 3 段实海试短视频（去鳞/剖切/真空），并给一份真实反馈表。达标即样机归您，并享首批量分销权。」——用一台样机换一条带货视频 + 一个渠道节点。

筛选标准：沿岸 charter、日接待 ≥6 客、有社媒（YouTube/Instagram/TikTok）≥5k 粉、船龄 <10 年（设备好装）。首批 10 人，美/欧/澳各 3–4 人。

  • Charter：「靠岸前把鱼处理好 → 多跑一趟、鱼更鲜、客人更满意、几趟回本」（ROI 驱动）。

八、成功指标（KPI）Success Metrics

  • 产品：通过 IP67 + 96h 盐雾 + 25° 横摇三项门槛；取得 ABYC/CE 报告。
  • 内容：船长种子计划产出 ≥10 条实海试视频，总播放 ≥50 万。
  • 渠道：首批量（200 台）上市 90 天内签约 ≥3 家海事分销商 / 1 家船厂 OEM 试点。
  • 商业：船版毛利率不低于陆地版同档；耗材（刀头/真空袋）复购贡献 ≥15% 船版收入。
