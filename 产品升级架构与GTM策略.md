# TailFin Marine — 模块化升级架构 · 双型号定价 · GTM 策略

> 生成日期：2026-07-17
> 关联文档：`产品设计方案文档.md`、`珠三角供应商匹配方案.md`
> 目标：让 **Standard 通过加购模块真实升级到 Pro**，理顺价格关系，并制定上市策略。

---

## 一、问题诊断：当前设计为什么无法升级

原方案里 Standard 与 Pro 是"两套不同产品"，而非"一套平台 + 加装模块"：

| 维度 | Standard（原） | Pro（原） | 升级障碍 |
|------|---------------|-----------|---------|
| 去鳞机 | 12V 100W 有刷 | 24V 200W 无刷 | 整机不同，无法"加装" |
| 电池 | 无（仅有线） | 内置 24V 锂电 | Standard 无电池舱，无处加装 |
| 中央仓 | 储物抽屉 | 真空密封模组 | 二者互斥，且抽屉≠可升级底座 |
| 鱼刀 | 无 | 12V 电动鱼刀 | 纯新增，无接口预留 |
| 供电 | 5m 硅胶线接点烟器 | 离网 2h | 无内部总线，离网无从谈起 |

**结论**：Standard 用户想变 Pro，等于"再买一台 Pro"，原有投入（箱体、去鳞机）几乎作废——这是产品与商业的双重失误。

## 二、新模块化架构：统一基础平台 + 双槽叠层中央仓

### 2.1 核心原则

> **一套基础平台（Base Platform）100% 共用；所有差异来自"插进中央仓的 cartridge"与"手持工具"。**
> Standard = 平台 + 储物抽屉 + 12V 有刷去鳞机；Pro = 平台 + 真空 cartridge + 电池 cartridge + 无刷去鳞机 + 电动鱼刀。

### 2.2 中央功能仓（Center Bay）双槽叠层设计 ⭐

这是升级可行性的关键工程改动：

- **物理**：箱体内腔预留标准仓室，深度容纳 **2 个标准 cartridge（下槽 + 上槽）**，总高约 13–14cm（在 26cm 箱高内，扣除砧板/底板后合理）。
- **电气**：每槽底部为公 Pogo Pin、顶部为母 Pogo Pin，形成**叠层供电/通信总线**；下槽连接箱体 PCBA 主供电（12V 外部输入 / 24V 内部电池）。
- **机械**：每槽两侧多瓣式快装卡扣轨道，推入即锁、拉环解锁抽出，**盲插（blind mate）**。
- **真空接口**：上槽额外预留真空腔软管引出至砧板排水槽旁的密封舱口。

**槽位分配**
| 状态 | 下槽（Lower） | 上槽（Upper） |
|------|--------------|--------------|
| Standard 出厂 | 电源母板挡板（占位） | 储物抽屉 Storage Drawer |
| Pro 出厂 | 电池 cartridge（24V 6S1P） | 真空密封 cartridge（≥-75kPa） |
| **升级路径** | 推入电池 cartridge | 抽掉抽屉 → 推入真空 cartridge |

### 2.3 三大接口（沿用原设计，明确分工）

| 接口 | 位置 | 用途 |
|------|------|------|
| **Pogo Pin 叠层总线** | 中央仓每槽 | cartridge 热插拔供电/通信（IP67） |
| **多瓣式快装卡扣** | 去鳞机/鱼刀手柄 | 刀头、鱼刀片一插即用 |
| **防水航空插（IP67）** | 箱体侧面 | 12V 外部供电 / 为电池充电 / 去鳞机·鱼刀取电 |

### 2.4 手持工具（非槽模块，但为升级加购项）

- 12V 100W 有刷去鳞机 — Standard 基线，接侧边 12V 航空插。
- 24V 200W 无刷去鳞机 — Pro / 升级项，由电池 24V 总线供电（离网）或外部 24V。
- 12V 往复式电动鱼刀 — Pro / 升级项，接侧边航空插。

### 2.5 模块清单（可独立销售，也是升级件）

| 模块 | 代号 | 说明 |
|------|------|------|
| 真空密封 cartridge | `VAC-MOD` | 干湿两用，≥-75kPa，含隐藏液体收集盒 |
| 电池 cartridge | `PWR-MOD` | 24V 6S1P ~50Wh，BMS，离网≥2h |
| 储物抽屉 | `DRW-MOD` | 基础收纳（Standard 出厂标配） |
| 24V 无刷去鳞机 | `SCL-24` | 性能升级 |
| 12V 电动鱼刀 | `KNF-12` | 增配 |
| 12V 有刷去鳞机 | `SCL-12` | 基线（Standard 出厂标配） |

## 三、Standard → Pro 升级路径（BOM 级证据）

**Standard 出厂配置**
```
Base Platform（箱体+水泵+HDPE砧板+316五金+PCBA+侧插+拉带槽）
+ DRW-MOD（储物抽屉）
+ SCL-12（12V 100W 有刷去鳞机，有线）
```

**Pro Upgrade Kit（加购套装）**
```
VAC-MOD（真空密封 cartridge）
+ PWR-MOD（电池 cartridge）
+ SCL-24（24V 200W 无刷去鳞机）
+ KNF-12（12V 电动鱼刀）
```

**升级后 = Pro 功能**
```
Base Platform（沿用）
+ VAC-MOD（上槽）+ PWR-MOD（下槽）   → 真空 + 离网
+ SCL-24（接侧边插，24V 总线）        → 无刷去鳞
+ KNF-12（接侧边插）                  → 电动鱼刀
（原 SCL-12、DRW-MOD 保留为备用/收纳）
```

✅ **逻辑闭环**：Standard 用户加购 4 个模块即获得 Pro 全部能力，原有箱体与备用去鳞机/抽屉不浪费。

## 四、双型号定价合理化

### 4.1 成本基准（ex-works 估算，千台级，待 RFQ 复核）

| 组件 | ex-works (¥) | 约合 ($@7.2) |
|------|-------------|-------------|
| Base Platform | 1,000 | 139 |
| DRW-MOD 储物抽屉 | 50 | 7 |
| SCL-12 有刷去鳞机 | 220 | 31 |
| VAC-MOD 真空 cartridge | 320 | 44 |
| PWR-MOD 电池 cartridge | 420 | 58 |
| SCL-24 无刷去鳞机 | 380 | 53 |
| KNF-12 电动鱼刀 | 150 | 21 |

### 4.2 型号与套装成本

| 项目 | BOM 构成 | ex-works (¥) | ex-works ($) |
|------|---------|-------------|-------------|
| **Standard** | Base + DRW + SCL-12 | 1,270 | 176 |
| **Pro Upgrade Kit** | VAC + PWR + SCL-24 + KNF | 1,270 | 176 |
| **Pro** | Base + VAC + PWR + SCL-24 + KNF | 2,270 | 315 |

> 注：Standard 升级到 Pro 后，用户实际拥有 = Standard + Kit = 成本 ¥2,540（多出的 ¥270 是原 SCL-12+DRW 作为备用保留）；而 Pro 整机成本仅 ¥2,270（不含冗余备件），故 **Pro 整机的毛利略高于"自购升级"**，两边皆健康。

### 4.3 零售定价（D2C，含 25% 关税+运费后的目标毛利 ~55–60%）

| 项目 | 单买模块价 ($) | 套装/型号价 ($) | 说明 |
|------|---------------|----------------|------|
| SCL-24 无刷去鳞机 | 129 | — | 单买 |
| KNF-12 电动鱼刀 | 79 | — | 单买 |
| VAC-MOD 真空 cartridge | 199 | — | 单买 |
| PWR-MOD 电池 cartridge | 149 | — | 单买 |
| **单买模块合计** | **556** | — | 散买无折扣 |
| **Pro Upgrade Kit** | — | **499** | 4 模块捆绑，较散买省 $57（≈10%） |
| **Standard** | — | **499** | 基线 |
| **Pro** | — | **999** | = Standard + Kit（$499+$499=$998≈$999） |

### 4.4 价格关系论证（为什么合理）

1. **对称成本 → 对称定价**：Standard 与 Kit 的 ex-works 成本完全相同（¥1,270），因此同价 $499 在成本结构上站得住。
2. **升级等式成立**：$499 + $499 = $998 ≈ Pro $999。用户"买 Standard 后加购 Kit"与"直接买 Pro"花费几乎一致，**升级路径经济可行、无惩罚**。
3. **捆绑激励**：散买 4 模块 $556，Kit $499，引导用户买 Kit（也利于 TailFin 一次性交付、降低履约成本）。
4. **Pro 整机微利优势**：Pro 不含冗余备件，成本低于"Standard+Kit 实际持有"，售价却持平，毛利更优——鼓励直接推 Pro，同时保留升级作为兜底转化。
5. **Mini 锚点**：$149 口袋去鳞机作为入门，可向上交叉销售 Standard 箱体（$499−$149 的差价即"加个工作站"）。

## 五、GTM 上市策略

### 5.1 信息屋（Messaging House）— 客户友好，无内部黑话

- **Hero Message**：*“Start with Standard. Grow into Pro. One box, endless upgrades.”*
  （中文：一套工作站，随你成长——今天 Standard，明天 Pro，从不重复花钱。）
- **支柱 1 · 升级不浪费**：买 Standard 不怕过时，加购模块即变 Pro，原有投入全保留。
- **支柱 2 · 车斗即工厂**：去鳞→冲洗→真空密封，在皮卡尾门一站式完成，干净省心。
- **支柱 3 · 北美原生适配**：26cm 低箱身，完美推入 Tonneau Cover / DECKED / BedSlide；316 海军钢抗海水。
- **产品级话术**：Standard=“钓完即清理的清爽闭环”；Pro=“移动鱼类加工厂，离网 2 小时”。

### 5.2 价格阶梯与升级变现

```
$149  Mini（入门手持）
  │  ↑ 交叉销售：加箱体变 Standard
$499  Standard（基础闭环）
  │  ↑ Pro Upgrade Kit $499（4 模块）
$999  Pro（移动加工厂）
```
- **升级触发点**：在 Standard 包装内附 DTC 二维码“一键升级 Pro”；湖边/码头现场看到 Pro 用户真空封鱼，即时转化。
- **复购耗材（高毛利、持续）**：防穿刺真空袋 $19/20 片、316 去鳞刀头 $29、PU 进水管 $25。

### 5.3 渠道策略

| 渠道 | 上架品类 | 角色 |
|------|---------|------|
| **Bass Pro / Cabela's 实体店** | Standard + Pro + 真空袋/刀头耗材 | 物理体验 + 阻断 Temu/白牌线上截流；买家可亲手插拔 cartridge |
| **DTC 官网**（已建） | 全系 + **Pro Upgrade Kit + 单模块** | 最高毛利、直接用户关系、升级转化主阵地 |
| **Amazon** | Mini + 耗材补充装 | 承接搜索流量与复购，不主推整机以免比价 |
| **户外电源/皮卡改装店**（DECKED/BedSlide 合作） | Standard/Pro 捆绑改装套餐 | 场景化联名，触达精准人群 |

### 5.4 分阶段上市节奏

| 阶段 | 时间 | 关键动作 |
|------|------|---------|
| **Phase 0 研发冻结** | M1–2 | 双槽中央仓模具定型、EVT/DVT、认证（FCC/UL/FDA/IPX6/UN38.3） |
| **Phase 1 DTC 软启动** | M3 | 官网首发 Standard + 配件；Catch&Cook 内容种草；建邮件名单 |
| **Phase 2 实体渠道突破** | M4–5 | Bass Pro/Cabela's  buyer 会议 + 上架；同步发布 Pro + Upgrade Kit |
| **Phase 3 规模化** | M6+ | 全品类 + 真空袋订阅制 + 钓手大使计划 + DECKED 联名 |

### 5.5 防 Temu / 白牌截流

1. **物理零售护城河**：Bass Pro/Cabela's 门店体验 + 导购，白牌无此触点。
2. **专属 cartridge 生态**：多瓣式快装卡扣 + 防穿刺真空袋为专用接口（客户侧表述为"完美贴合、锁鲜不漏"），白牌耗材不兼容。
3. **升级关系绑定**：Standard 用户已是生态内，Kit/耗材复购自然回流 DTC。

### 5.6 关键 KPI

- 升级转化率：Standard 买家 12 个月内购 Kit 比例（目标 ≥18%）
- DTC 毛利率（目标 ≥58%）
- 耗材复购率（订阅制目标月留存 ≥65%）
- 实体店连带率：Pro/Kit + 耗材同篮比例
