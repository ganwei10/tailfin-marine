## 1. 产品概述

### 1.1 产品定位

| 项目 | 内容 |
|------|------|
| 产品名称 | 卡车尾门移动鱼类加工工作站 |
| 英文名 | Truck Tailgate Mobile Fish Processing Station |
| 品牌 | TailFin Marine |
| 目标市场 | 美国、加拿大 |
| 核心受众 | 皮卡车/SUV车主、硬核路亚/海钓发烧友、冰钓/徒步 Catch & Cook 钓客 |
| 价格区间 | $149 — $999 |

### 1.2 市场痛点与解决方案

| 痛点 | 现状 | TailFin Marine 方案 |
|------|------|---------------------|
| 场景断层 | 公共码头洗鱼台排队久、蚊虫多、卫生差 | 车斗即加工站，随时随地去鳞切割 |
| 环境污染 | 鱼鳞飞溅、血水粘附车斗极难清洗 | 封闭式砧板导流槽 + IPX6 高压冲洗 |
| 冷链保鲜 | 血水融化导致鱼肉二次污染变质 | 就地真空密封，锁鲜至家 |
| 供电受限 | 户外无市电，加工设备无法使用 | 12V/24V 车载/电池双制式供电 |

### 1.3 三档产品矩阵

| 档位 | 产品名 | 售价 | 定位 | 核心差异 |
|------|--------|------|------|----------|
| Good | The Pocket Scaler (Mini) | $149 | 徒步/皮划艇轻量钓客 | 仅手持去鳞机+配件包，无箱体 |
| Better | Tailgate Workstation (Standard) | $499 | 大众皮卡车主 | 完整箱体+去鳞+给水+**储物抽屉仓** |
| Best | Mobile Fish Factory (Pro) | $999 | 重度发烧友/海钓客 | 无刷去鳞+真空密封+电池+鱼刀 |
| — | **Pro 升级套装 (Upgrade Kit)** | **$499** | Standard  upgraders | 4 模块捆绑，使 Standard → Pro |

> **升级闭环（V2 核心变更）**：Standard 与 Pro 共用同一基础平台，差异仅在于「中央功能仓插拔模块」与「动力/电池配置」。Standard 用户购买 **Pro 升级套装（$499）** 即可原地升级到 Pro，原有箱体与有线去鳞机保留作备用——升级零浪费。Standard($499) + Kit($499) = $998 ≈ Pro($999)，价格对称见 §6.2。

## 2. 工业设计 (ID)

### 2.1 整体造型语言

- **风格定位**: 工业战术风 (Industrial Tactical) — 厚重、硬朗、可信赖
- **色彩方案**: 
  - 主色: 深海蓝 (Navy #0a2342) — 专业感
  - 辅色: 海洋绿 (Teal #0F6E56) — 品牌识别
  - 点缀: 琥珀橙 (Amber #EF9F27) — 能量/警示
- **表面处理**: 箱体外壳哑光磨砂纹理 (Orange Peel)，防刮防滑

### 2.3 关键尺寸约束

| 参数 | 数值 | 设计理由 |
|------|------|----------|
| 长度 | 80 cm | 适配标准皮卡车斗宽度，留出两侧余量 |
| 宽度 | 44 cm | 单人可搬运，适配 DECKED 抽屉宽度 |
| **高度** | **≤ 26 cm** | **核心约束** — 确保推入 Tonneau Cover / DECKED 抽屉不干涉 |
| 重量(空) | ≤ 12 kg (Standard) / ≤ 15 kg (Pro) | 单人可装卸 |

### 2.4 人机工程

- **把手**: 箱体正面中央嵌入式把手，深度 4cm，宽度 40cm，戴手套可握
- **砧板翻折**: 双铰链设计，可向前/向后翻折180°，适应车斗正反两面操作
- **操作高度**: 箱体置于尾门上时，砧板面距地面约 90cm，符合站立切鱼 ergonomics
- **线缆管理**: 所有线缆均有专用收纳槽，闭合箱盖自动收线

> **关键变更**：V1 中 Standard 与 Pro 是两套互相排斥的产品（去鳞机有刷/无刷、有无电池、抽屉/真空全部互斥），Standard 用户无法升级。V2 改为 **统一基础平台 + 中央仓双槽叠层**，Standard→Pro 仅通过插拔 4 个模块完成。

### 4.2 中央功能仓 — 双槽叠层插拔机构（核心）

**结构**: 仓体分上下两槽（U/L），每层独立 Pogo Pin 电气接口 + 多瓣式快装卡扣机械锁，支持盲插与叠放。

| 槽位 | Standard 出厂 | Pro 出厂 / Upgrade Kit |
|------|---------------|------------------------|
| 上槽 (U) | 注塑工具抽屉 (无电气) | 真空密封 cartridge（湿/干两用，≥-75kPa，含液体收集盒） |
| 下槽 (L) | 空位（带占位盖板，无电气） | 24V 6S1P 锂电池 cartridge（带 BMS，Pogo Pin 接入） |

**升级动作**：抽出抽屉 → 上槽推入真空 cartridge → 下槽推入电池 cartridge → 配无刷去鳞机 + 电动鱼刀。原有有线去鳞机保留备用。

**Pogo Pin 规格**:
- 触点: 4 Pin (VCC + GND + TX + RX)，额定电流 5A/pin
- 插拔寿命: ≥ 10,000 次；对接后防护 IP67
- 盲插: 导轨 + 极性防呆，用户无需工具

### 4.3 去鳞机模块 — 共用核心

| 参数 | Mini | Standard | Pro / Upgrade Kit |
|------|------|----------|-----|
| 外壳模具 | **共用** Pro 外壳模具 | 箱体集成供电 | **基准模具** |
| 刀头 | **共用** 316 SS 滚齿 | **共用** | **共用** |
| 电机 | 12V 有刷 100W | 12V 有刷 100W | 24V 无刷 200W |
| 供电 | 内置 2200mAh 电池 | 箱体有线 (12V) | 箱体电池/有线 (24V) |
| 防溅罩 | **共用** PC 罩 | **共用** | **共用** |
| 快装接口 | **共用** 多瓣式卡扣 | **共用** | **共用** |

> **核心复用**: 去鳞机外壳、316 SS 刀头、PC 防溅罩、多瓣式快装卡扣 — 四部件 100% 共用，仅电机与供电方式不同。Standard 的有刷去鳞机与 Pro 的无刷去鳞机为可互换手持件。

### 4.4 专用耗材与卡扣锁定

| 耗材 | 规格 | 接口锁定机制 |
|------|------|----------|
| 去鳞刀头 | 316 SS 滚齿 | 多瓣式快装卡扣 (非标准接口) |
| 动力鱼刀片 | 12V 往复式刀片 | 专用卡扣 (与去鳞头共用接口) |
| 真空密封袋 | 防鱼刺穿刺 | 专用袋口尺寸 + 密封条纹路 |

> 该设计使耗材与替换件形成长期复购闭环，并通过非标接口保障用户体验一致性。

## 5. 电气与电控设计

### 5.1 供电架构

```
Standard 款供电链:

  卡车点烟器 12V
       │
       │ 5m 14AWG 硅胶线
       ▼
  ┌─────────┐
  │ PCBA    │── 12V → 水泵
  │ 主板    │── 12V → 去鳞机 (有刷)
  │ (灌胶)  │── 12V → 照明LED (可选)
  └─────────┘


Pro 款供电链:

  ┌──────────────────┐
  │ 24V 6S1P 电池包  │
  │ 50Wh + BMS       │
  └────────┬─────────┘
           │
     ┌─────┴─────┐
     │  PCBA     │── 24V → 去鳞机 (无刷)
     │  主板     │── 12V → 动力鱼刀 (DC-DC 降压)
     │ (灌胶)    │── 12V → 水泵
     │           │── 24V → 真空密封模块
     └───────────┘
           │
     ┌─────┴─────┐
     │ 充电输入  │── 110V AC → 整流 → BMS
     │           │── 12V DC → BMS
     └───────────┘
```

### 5.2 PCBA 主板设计要求

| 项目 | 要求 |
|------|------|
| 防水工艺 | 全面灌胶 (Potting), 硅胶灌封, 厚度 ≥ 3mm |
| BMS 功能 | 过充/过放/过流/短路保护, 温度监测 (NTC) |
| 充电管理 | CC/CV 充电曲线, 支持 110V AC (Pro) / 12V DC (Std/Pro) |
| DC-DC | 24V→12V 降压 (Pro), 效率 ≥ 90% |
| 过载保护 | 水泵/去鳞机独立保险丝 + MCU 检测卡死电流 |
| 重置方式 | 一键 Reset 按钮 (箱体侧面, IP67 防水) |

### 5.3 电机与动力参数

| 设备 | 电压 | 功率 | 类型 | 保护 |
|------|------|------|------|------|
| 去鳞机 (Std) | 12V | 100W | 有刷 | 卡死断电 + Reset |
| 去鳞机 (Pro) | 24V | 200W | 无刷 | 卡死断电 + Reset |
| 动力鱼刀 (Pro) | 12V | 80W | 往复式 | 过热断电 |
| 水泵 | 12V | 18W | 隔膜自吸 | 防砂/干转断电 |
| 真空泵 (Pro) | 24V | 40W | 活塞式 | 液体收集盒保护 |

### 5.4 开关与安全

- **双击启动**: 所有高频高扭矩电动开关 (去鳞机、鱼刀) 必须双击启动
- **物理保险扣**: 去鳞机扳机上方设物理保险扣 (需拇指推开后方可扣扳机)
- **防误触测试**: 通过 4G 随机振动 30min, 开关不误触发 (MIL-STD-810G)

## 7. 安全与合规要求

### 7.1 认证清单

| 认证 | 标准 | 范围 | 优先级 |
|------|------|------|--------|
| FCC | FCC Part 15B | 无刷电机 EMC | P0 |
| UL | UL 2054 / UL 60730 | 电池包+电控 | P0 |
| FDA | 21 CFR 177.1520 | HDPE/ABS 食品接触 | P0 |
| IPX6 | IEC 60529 | 箱体防水 | P0 |
| Salt Spray | GB/T 10125 / ASTM B117 | 316 SS 五金 | P1 |
| 加州 PROP 65 | — | 铅/镉/邻苯 | P0 (加州销售必须) |
| DOT | 49 CFR 173.185 | 锂电池运输 | P0 |

### 7.2 食品安全

- 所有接触鱼肉的表面 (砧板、导流槽) 必须通过 FDA 食品接触认证
- 真空密封袋必须通过 FDA 21 CFR 177.1520 (聚烯烃食品接触)
- HDPE 砧板不得含有 BPA 或邻苯二甲酸酯
- ABS+PC 箱体内层必须使用食品级树脂牌号

### 7.3 电池安全

- 6S1P 锂电池包必须通过 UN38.3 运输测试
- BMS 必须具备: 过充保护 (4.25V/cell)、过放保护 (2.75V/cell)、过流保护 (15A)、温度保护 (60°C 断开)
- 电池包必须独立封装在金属盒内, 通过 1m 跌落测试不变形
- 运输状态电池电量 ≤ 30%
